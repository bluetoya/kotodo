### kotodo 프로젝트 설명
코틀린을 학습하기 위해 To-Do 리스트 서비스 백엔드를 작성

### 사용한 기술
- Kotlin
- Spring Boot (3.2.5)
- MySQL
- JPA
이후 더 필요하면 추가 예정

### API 구현
- 할 일(task)
  - task 리스트 가져오기
  - task 생성
  - task 수정
  - task 삭제
- task 그룹
  - 그룹 전체 조회
  - 그룹 낱개로 조회
  - 그룹 생성
  - 그룹 수정
  - 그룹 삭제
- 루틴
  - 루틴 조회
  - 루틴 생성
  - 루틴 수정
  - 루틴 삭제
- 유저
  - 유저 조회
  - 유저 생성
  - 유저 정보 수정
  - 유저 삭제
