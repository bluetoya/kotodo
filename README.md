### kotodo 프로젝트 설명
코틀린을 학습하기 위해 To-Do 리스트 서비스 백엔드를 작성

### 사용한 기술
- 코틀린
- Spring Boot
- MySQL
- JPA
이후 더 필요하면 추가 예정

### (우선) 구현할 API
- tod 리스트 가져오기
- todo 추가
- todo 수정
- todo 삭제
